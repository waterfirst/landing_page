회의 요약

이번 모임에서는 AI 도구를 활용한 다양한 사례와 최신 동향을 공유했습니다. 다양한 주제로 진행되었으며, 특히 GPT와 클로드 등 AI 모델을 활용한 실제 업무 적용 사례, 커서 AI와 같은 코딩 도구, MCP(Model Context Protocol) 연결을 통한 확장 기능, 뉴스 크롤링 및 모니터링 시스템 구축, 깃허브 연동, 클라우드 서비스 활용 등 폭넓은 논의가 이루어졌습니다.

1. GPT를 활용한 곡률 스트레스 시뮬레이션 사례

  • 곡면 디스플레이 개발 프로젝트 사례 공유
    • 전자 MX에서 중심부는 평평하고 모서리가 둥근 돔 형태의 디스플레이 개발을 요청받음
    • 스트레처블 백플레인이나 다른 고려 요소 없이 기존 패널을 구부리는 방식 검토 필요
    • 회사 내 CAE팀에 의뢰할 경우 승인 및 일정 소요가 길어 개인적으로 GPT를 활용한 시뮬레이션 진행
  • GPT 활용 방식
    • 프롬프트 작성 방법: 업무 지시보다 고민 상담 형태로 접근하여 더 자연스러운 대화 유도
    • 코드 작성: R 언어를 사용한 시각화 코드를 GPT가 생성
    • GUI 기반 시각화 도구 생성: 곡률, 높이, 패널 크기 등을 조절할 수 있는 인터랙티브 도구 구현
  • 시뮬레이션 결과 분석
    • 곡률에 따른 응력 분포를 히트맵 형태로 시각화
    • 모서리 부분과 코너에 집중되는 응력 패턴 확인
    • 주름 형태 예측 및 곡률에 따른 스트레스 변화 시각화
    • 곡률이 심할수록 단면에 주름이 심해지는 현상 시각화
  • 응용 및 효과
    • 금형 설계 방향성 검토 및 시각화
    • 스트레스 집중 부분의 절개를 통한 디자인 개선 방안 도출
    • PI 필름 등 실제 제작 방법에 대한 아이디어 시각화
    • 회사 보고용 문서 및 발표 자료 자동 생성

2. 클로드 AI 활용 사례 - MCP 연결 및 뉴스 크롤링 시스템

  • 글로벌 뉴스 모니터링 시스템 구축
    • 다양한 뉴스 소스에서 데이터를 수집하여 텔레그램으로 전송하는 시스템 개발
    • 텔레그램 API 연동 및 봇 설정을 통한 메시지 자동 전송 구현
    • 일정 시간마다 자동으로 뉴스를 수집하여 번역 후 텔레그램으로 전송
    • 구글 번역 API와 제미나이 모델을 활용한 한글 요약 기능 구현
  • MCP(Model Context Protocol) 활용
    • 엔트로픽에서 개발한 MCP를 통해 다양한 서비스와 AI 연동
    • 클로드 데스크톱 버전을 통해 로컬 파일 검색 및 관리 기능 구현
    • 브레이브 서치, 퍼플렉시티 등 검색 엔진 MCP를 통한 최신 정보 검색
    • 깃허브 MCP를 활용한 코드 관리 및 배포 자동화
  • Streamlit을 활용한 웹 인터페이스 구현
    • Python의 Streamlit 라이브러리를 활용한 웹 대시보드 구현
    • 수집된 뉴스 데이터의 시각화 및 필터링 기능 구현
    • 카테고리별(미국 경제, 중국 경제, 세계 정치, AI 기술, 투자 정보) 뉴스 분류 및 모니터링
  • 클로드와 GPT의 차이점 및 특징
    • 클로드: 코드 작성 능력이 우수하며, 대화의 맥락 이해도가 높음
    • GPT: 대화의 연속성 유지에 강점, 긴 대화에서도 안정적인 성능
    • 클로드는 감성적인 대화와 상담에 더 적합하다는 의견 공유

3. 커서 AI와 최신 코딩 도구 소개

  • 커서 AI 소개 및 특징
    • VS Code와 유사한 인터페이스에 AI를 접목한 코딩 도구
    • 자연어로 코드 작성이 가능한 "바이브 코딩" 방식 지원
    • 코드 자동 완성 및 추천 기능 제공
    • 캡처한 웹사이트 이미지를 기반으로 유사한 웹페이지 자동 생성 가능
  • API 연동 및 확장 기능
    • 딥식, 제미나이 등 다양한 AI 모델 API 연동 가능
    • 터미널 명령어 자동 실행 및 추천 기능
    • 깃허브 자동 연동 및 코드 푸시 자동화
  • AI 코딩 도구의 현황과 전망
    • 오픈 AI가 커서 AI 인수 시도 (30억 달러 규모)
    • 윈드 서프 등 경쟁 서비스의 등장
    • 코딩 생산성 향상으로 개발자 역할 변화 예상
    • 기획력과 AI 활용 능력이 중요해지는 개발 환경 변화

4. API 사용 방법과 클라우드 서비스 활용

  • 다양한 API 활용 사례
    • 텔레그램 API: 봇 생성 및 메시지 자동 전송 (토큰과 ID 필요)
    • 구글 번역 API: 다국어 뉴스 번역 및 요약
    • 제미나이 API: 텍스트 요약 및 생성 (무료 제공)
    • AI 서비스 API 요금제: 정액제 vs 종량제
  • 깃허브와 클라우드 서비스 활용
    • 깃허브: 코드 버전 관리 및 간단한 웹사이트 호스팅 (무료 퍼블릭 저장소)
    • AWS(Amazon Web Services): 24시간 서비스 운영을 위한 서버 호스팅
    • AWS 자격증 언급 (2년 유효, 13만원 시험비)
    • 클라우드 서비스의 비용 효율성과 확장성
  • 서버와 깃허브의 차이점
    • 깃허브: 주로 코드 관리와 정적 웹사이트에 적합
    • 클라우드 서버: 동적 서비스 및 지속적 운영에 필요
    • 클라우드 서비스 선택 기준: AWS, 구글 클라우드, 네이버 클라우드 등

5. AI 기술의 교육 및 실무 활용

  • AI를 활용한 코딩 교육
    • 초등학교 수학 문제 해결에 GPT 활용한 사례 공유
    • AI가 직관적인 풀이 방법 제안 및 연습 문제 생성
    • 프로그래밍 입문자를 위한 도구로서의 가능성
  • AI를 활용한 업무 생산성 향상
    • 회사 주간 보고서 및 발표 자료 자동 생성
    • 대용량 매뉴얼(3,000~4,000페이지) 검색 및 질의응답 시스템
    • 특허 출원 과정 자동화 (명세서 작성 및 도면 생성)
  • AI 활용의 균형점
    • 스마트폰의 대중화처럼 AI 도구의 대중화 예상
    • AI 활용 능력에 따른 생산성 격차 발생
    • 기획력과 AI 활용 능력의 조화 필요성

6. AI 기술 동향 및 서비스 비교

  • 주요 AI 모델 및 기업 현황
    • OpenAI(GPT): MS 투자, 선도적 위치
    • Anthropic(Claude): 아마존 투자, 코드 작성 및 감성적 대화에 강점
    • Google(Gemini): 뒤늦게 따라잡기, 무료 API 제공으로 경쟁
    • xAI(Grok): 엘론 머스크의 회사, 1년 만에 참여
    • 중국의 TC, Meta의 Llama 등 오픈소스 진영
  • AI 서비스 요금제 및 사용 팁
    • 정액제: 월 20불로 제한적 사용 가능
    • API 사용: 사용량에 따른 종량제 요금
    • 무료 서비스: Gemini는 API도 무료로 제공 중
    • 사용량 제한: 과도한 사용 시 일시적 제한 발생
  • 현재 추세 및 발전 방향
    • 3개월마다 급격한 기술 변화 발생
    • MCP를 통한 서비스 연동 가속화
    • 2차 서비스 업체들의 등장 (특화된 영역별 서비스)
